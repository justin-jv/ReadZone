# 📖 About the Project

ReadZone is a full-stack e-commerce web application designed for purchasing books online. It provides customers with a smooth shopping experience while allowing administrators to efficiently manage books, categories, users, orders, coupons, offers, and other store operations through a dedicated admin panel.

The project focuses on building a secure, responsive, and user-friendly bookstore using Django's MVT architecture and PostgreSQL database.
